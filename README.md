# Amazon E-Commerce Sales Dashboard using Excel

## Project Overview
This project shows an Amazon E-Commerce Sales Dashboard created using Microsoft Excel.

The dashboard helps to analyze total sales, total orders, total quantity, cancelled orders, top category, monthly sales trend, category-wise sales, order status, customer type, and state-wise sales.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- KPI Cards
- Slicers
- Data Cleaning
- Dashboard Design
- Business Insights

## Dataset
The dataset contains Amazon e-commerce sales data with details like order ID, order date, product category, order status, order type, quantity, amount, and shipping state.

## Work Done
- Cleaned the raw dataset
- Created a cleaned data sheet
- Created Pivot Tables
- Created KPI Cards
- Created Pivot Charts
- Added slicers for interactive filtering
- Built a clean Excel dashboard layout
- Added key business insights

- Top Category: Set
- Top State: Maharashtra

## Key Insights
- Total sales performance was ₹7.85 crore.
- April recorded the highest monthly sales.
- Set was the top-selling category.
- Most orders came from B2C customers.
- Successful orders were higher than cancelled orders.
- Maharashtra generated the highest state-wise sales.

## Recommendations
- Focus more on high-performing categories like Set, Kurta, and Western Dress.
- Reduce cancelled orders to improve revenue and customer experience.
- Target top-performing states like Maharashtra and Karnataka for better sales growth.
- Monitor monthly sales trend to understand sales decline after April.
